# Interactive Personal Data Collector

A beginner-friendly Python script that collects personal information from the user, processes it, and displays the data along with some interesting "under-the-hood" technical details. 

This project is a great demonstration of basic Python concepts, including user input, data type casting, and understanding how Python stores variables in memory.

## 🚀 Features

* **Interactive Prompts:** Asks the user for their name, age, height, and favorite number.
* **Data Type & Memory Inspection:** Uses built-in Python functions to reveal the data type (`type()`) and the specific memory address (`id()`) where each piece of information is stored.
* **Basic Calculations:** Computes an approximate birth year based on the current year (2026) and the provided age.
* **Type Casting:** Demonstrates converting a floating-point number (height) into an integer for an approximate value.
* **Summary Output:** Consolidates all the user's data into a clean, easy-to-read final sentence.

## 🛠️ Concepts Covered

If you are learning Python, this script touches on several fundamental concepts:
* `input()` and `print()` functions
* Data types (`str`, `int`, `float`)
* Type casting (converting between data types)
* The `type()` function
* The `id()` function for memory management awareness
* Basic mathematical operations
